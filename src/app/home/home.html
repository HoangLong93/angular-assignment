Angular Assignment

Your mission, if you choose to accept it, is following:

Implement new component with a view that displays contact information as business-card like boxes.

Requirements:

  • Component code into /src/app/cards + possible changes to existing files
  • Router-url #/cards, add view to app main menu as "Cards"
  • Follow the structure of existing components when ever possible possible
  • See existing components on how to fetch contact data
  • Try to use AngularJS built in features as much as possible
  • Card layout is free, try to use Bootstrap styles
  • Cards should "flow" from left to right and downwards
  • All visible cards should be same size, amount of cards on one row should depend on viewport width
  • Show only contacts which are "active"
  • Implement a search filter which matches to contact fields and shows only matching contacts
